===================
Inertia Middleware
===================

Middleware for handling Inertia.js protocol requirements.

The InertiaMiddleware automatically:
- Detects Inertia requests via X-Inertia header
- Manages version negotiation
- Handles partial reloads
- Processes deferred and merge props
- Sets appropriate response headers
